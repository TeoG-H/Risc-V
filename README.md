# RISC-V Processor Implementation in Verilog

## Single-Cycle and Multi-Cycle Architectures 

This repository contains two independent implementations of a 32-bit RISC-V processor written in Verilog:
 - Single-Cycle CPU
 - Multi-Cycle CPU

Both designs implement a subset of the RV32I instruction set and are fully simulated and functionally verified.

# Each implementation contains:
- Complete datapath design
- Control unit logic
- Instruction and data memory
- Functional testbench
- Simulation waveforms
